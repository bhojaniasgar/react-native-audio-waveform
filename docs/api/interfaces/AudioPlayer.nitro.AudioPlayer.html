    Interface AudioPlayer

    AudioPlayer Hybrid Object

    Provides audio playback functionality with precise control over playback state, speed, and volume via direct JSI integration. Supports multiple concurrent players (up to 30).

    Example

    const player = audioWaveform.createPlayer('player1')

    // Prepare the player
    await player.prepare({
      path: '/path/to/audio.mp3',
      volume: 0.8,
      updateFrequency: UpdateFrequency.Medium
    })

    // Monitor playback progress
    player.onPlaybackUpdate((position) => {
      console.log('Current position:', position, 'ms')
    })

    // Start playback
    await player.start(0, 1.0)

    // Seek to position
    await player.seekTo(30000) // 30 seconds

    // Stop playback
    await player.stop()
